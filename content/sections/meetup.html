Tom Sydney Kerckhove - Cooperative distributed agreement

Thursday 28 November, 19:00

I will talk about mergeful [1]: A library for synchronising data across
clients via a central server. It handles collaborative agreement of a collection of items with safe merge conflicts.

This work originated in my work on Smos [2], a comprehensive "Getting Things Done" system in a terminal user interface, that now supports synchronisation across devices. You can read more about it on my blog [3], where I have a multi-part blog post series about mergeful.

[1] https://github.com/NorfairKing/mergeful
[2] https://smos.cs-syd.eu
[3] https://cs-syd.eu
